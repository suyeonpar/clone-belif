### * 내가 이 사이트를 선정한 이유

---
해당 사이트를 선정한 이유는 첫번째, 복잡한 디자인 보다는
한눈에 알아보기 쉬운, 심플하면서 사이트의 주제가 확실히 드러나는 것을 
추구하기 때문입니다.

깔끔하면서 심플한 디자인과 이러한 디자인이 자칫 단조로워 보일 수 있기에 
글자에 움직이는 모션을 주어 심플하면서도 포인트를 준 것이 마음에 들었습니다.
